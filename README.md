# AssetFlow – Enterprise Asset Management System

AssetFlow is a simple and responsive **Enterprise Asset Management System** built using **HTML, CSS, and JavaScript**. It helps organizations manage assets, employees, departments, maintenance, bookings, and reports through an easy-to-use interface.

## Features

- Secure Login System
- Admin & Employee Accounts
- Dashboard with Statistics
- Asset Management
- Organization Setup
- Department Management
- Employee Management
- Asset Categories
- Asset Allocation & Transfer
- Resource Booking
- Maintenance Tracking
- Audit Management
- Reports & Analytics
- Notifications
- Dark, Light & System Theme
- Responsive Design
